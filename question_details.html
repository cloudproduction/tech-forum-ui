You should retrieve RecyclerView in a Fragment after inflating core View using that View. Perhaps it can't find your recycler because it's not part of Activity

You’ll have to create separate Java classes for each fragment (setting the layout of each class to the corresponding xml file) and then use these classes in your main activity file, Just create a different layout for each fragment and add a recycler view in xml file for whichever fragments you require.

Add this code in your MyAdapter

 public class MyAdapter extends RecyclerView.Adapter"MyAdapter.ViewHolder" {
                        private ItemData[] itemsData;
                        public MyAdapter(ItemData[] itemsData) {
                        this.itemsData = itemsData;
                        }

So the main thing here is that anywhere you call findViewById() you will need to use 

rootView.findViewById()
